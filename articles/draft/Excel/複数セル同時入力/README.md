# 複数セル同時入力
## 概要	
複数のセルに同じデータを同時に入力する方法を紹介します。

## 事例	
以下のように、選択している複数のセルに同じ内容を入力したいとき、コピー＆ペーストでは少し面倒です。  
![](image1.png)

## 解決方法	
以下の操作により、同じ内容を複数セルに同時入力できます。

1. ShiftやCtrlを押しながら、入力したいセルを選択します。  
![](image1.png)

2. [F2]キーを押してデータを入力します。  
![](image2.png)

3. [Ctrl + Enter]を押すと、選択した全セルにデータが入力されます。  
![](image3.png)

## 対応バージョン
Excel 2013で動作確認済みです。
