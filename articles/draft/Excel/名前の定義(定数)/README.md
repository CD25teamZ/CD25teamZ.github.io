# 名前の定義(定数)

## 概要
定数に名前を定義すると、定数を名前で参照することができます。

## 操作方法

[数式]タブより[名前の管理]を選択します。

![](image1.png)

[名前の管理]ダイアログが表示されたら、[新規作成]ボタンを押下します。

![](image2.png)

新しい名前を定義します。ここでは以下のように設定します。

|名前  |参照範囲   |
|-     |-          |
|地球半径|`=6371`|

入力したら[OK]を押します。

![](image3.png)

[閉じる]を押します。

![](image4.png)

定義した名前「地球半径」を以下のように参照すると、定義した値が表示されます。

![](image5.png)
